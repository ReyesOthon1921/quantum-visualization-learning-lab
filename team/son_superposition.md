\## What Is Superposition?



Superposition is one of the most important ideas in quantum computing.



A classical bit can only be:



\- 0

\- 1



A qubit can be:



\- 0

\- 1

\- a combination of both at the same time



This combination is called superposition.



\## Mathematical Representation



A qubit in superposition can be written as:



|ψ⟩ = a|0⟩ + b|1⟩



where:



a = amplitude for state |0⟩



b = amplitude for state |1⟩



The amplitudes determine the probability of measuring each state.



\## Equal Superposition



One common state is:



|+⟩ = (|0⟩ + |1⟩)/√2



This means:



Probability of measuring 0 = 50%



Probability of measuring 1 = 50%



Before measurement, the qubit behaves as if both possibilities exist.



\## How Superposition Is Created



The Hadamard Gate (H) creates superposition.



Example:



Initial State:



|0⟩



Apply H Gate:



H|0⟩ = (|0⟩ + |1⟩)/√2



The qubit is now in superposition.



\## Why Superposition Matters



Superposition allows quantum computers to represent many possibilities at once.



This helps quantum algorithms explore multiple computational paths simultaneously.



Superposition is one of the foundations of:



\- Quantum Parallelism

\- Quantum Algorithms

\- Entanglement

\- Quantum Teleportation



\## Example

\## Vocabulary



Superposition = multiple possibilities existing before measurement



Hadamard Gate = gate that creates superposition



Amplitude = value used to calculate probability



Measurement = observing a qubit and collapsing its state



Qubit = basic unit of quantum information
