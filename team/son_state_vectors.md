\## What I Learned



A qubit is different from a classical bit.



A classical bit can only be:



\- 0

\- 1



A qubit can be in a combination of both states at the same time.



This is represented by a state vector.



\## Single Qubit State



A quantum state can be written as:



|ψ⟩ = a|0⟩ + b|1⟩



where:



a = amplitude of state |0⟩



b = amplitude of state |1⟩



The amplitudes determine the probability of measuring each state.



\## Probability Rule



The probabilities must add up to 1.



|a|² + |b|² = 1



Example:



a = 1/√2



b = 1/√2



Probability of measuring 0:



50%



Probability of measuring 1:



50%



\## State Vector Form



The state:



|ψ⟩ = a|0⟩ + b|1⟩



can be represented as:



\[a]

\[b]



This is called a state vector.



\## Two-Qubit Systems



Two qubits can represent four basis states:



|00⟩



|01⟩



|10⟩



|11⟩



The state is written as:



|ψ⟩ = c0|00⟩ + c1|01⟩ + c2|10⟩ + c3|11⟩



The amplitudes determine the probability of measuring each state.



\## Why This Matters



State vectors are the mathematical language of quantum computing.



They allow us to describe:



\- superposition

\- entanglement

\- quantum gates

\- measurement



Understanding state vectors helps explain how quantum circuits work.



\## Vocabulary



Qubit = quantum information unit



Amplitude = coefficient that determines probability



State Vector = mathematical description of a quantum state



Basis State = possible measurement outcome



Probability = likelihood of measuring a state
