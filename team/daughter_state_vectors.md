\# State Vectors



A state vector is a way to describe a qubit.



Scientists use state vectors to keep track of quantum states.



State vectors help us understand:



\- qubits

\- superposition

\- measurement



A state vector stores information about the possible states of a qubit.



Today I learned that state vectors are used to describe quantum information mathematically.



\## Vocabulary



Qubit = the basic unit of quantum information



State Vector = a mathematical description of a quantum state



Measurement = observing a qubit and getting an answer



Superposition = a qubit being in multiple possibilities before measurement

Today I learned that quantum computers use state vectors to describe qubits and predict measurement outcomes.
